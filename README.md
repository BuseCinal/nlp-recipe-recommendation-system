# NLP Recipe Recommendation System

A Python-based recipe recommendation system that uses **Natural Language Processing (NLP)** techniques to suggest similar recipes based on their ingredients.

## Features

- Recommends recipes based on ingredient similarity.
- Uses **TF-IDF vectorization** and **cosine similarity**.
- Handles missing ingredient data gracefully.
- Case-insensitive recipe matching for user input.
- Simple interactive command-line interface.
